Enhancements:
    Visual enhancements
        - mouse icons for specific modes
    Options
        - allows different colors
        - different boldness sizes

To lasso select, click third button on toolbar
	- to transition, after selecting click inside the dashed selection area, while holding down drag the mouse
To add a frame click farthest right button on toolbar

To run
        $ make run
To clean
        $ make clean
